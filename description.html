Loki Pricing Table Generator

This is a web app that generate HTML and CSS code from 6 customizable pricing table designs. T he generated code can be copied to your theme, template or website and it will be compatible down to IE11.

You don't have to worry manually editing the HTML and CSS for customization..

FEATURES

  • Real-time style editor
  • Real-time content editor
  • Simple and easy to use
  • 6 pricing table designs
  • Color, borders, roundness customization and many more

GENERATED CODE

  • Responsive
  • Cross browser compatible
  • Do not depend on bootstrap and other frameworks
  • Clean and lightweight (no code bloat)
  • W3C validated
  • Can be copied to clipboard or save as file

IMPORTANT NOTES

  • This is not a WordPress Plugin or any other CMS plugin
  • The images shown in the preview is for demo purposes only and not included in the download package
